# NoFee Merchant App

App για τα μαγαζιά στο NoFee platform.

## Χαρακτηριστικά

- 📊 Dashboard με στατιστικά και πρόσφατες παραγγελίες
- 📦 Διαχείριση Παραγγελιών - προβολή, ενημέρωση status
- 🍕 Διαχείριση Προϊόντων - προσθήκη, επεξεργασία, διαγραφή
- 👤 Προφίλ Μαγαζιού - ρυθμίσεις και πληροφορίες

## Setup

```bash
npm install
```

## Run

```bash
npm start
```

## Screens

### Dashboard
- Στατιστικά (έσοδα, παραγγελίες, εκκρεμείς)
- Πρόσφατες παραγγελίες

### Orders
- Λίστα όλων των παραγγελιών
- Φίλτρα ανά status
- Ενημέρωση status παραγγελίας
- Προβολή λεπτομερειών παραγγελίας

### Products
- Λίστα προϊόντων
- Προσθήκη/Επεξεργασία προϊόντων
- Ενεργοποίηση/Απενεργοποίηση διαθεσιμότητας
- Αναζήτηση και φίλτρα ανά κατηγορία

### Profile
- Πληροφορίες μαγαζιού
- Ρυθμίσεις
- Βοήθεια

## Technologies

- React Native
- Expo
- TypeScript
- React Navigation (Bottom Tabs)
- Context API για state management
